# 🔬 半导体器件 Stack 图生成器 — 使用说明

## 概述

单文件 HTML 工具，双击在浏览器中打开即可使用。用于生成半导体器件截面结构示意图（Stack Diagram），输出为 SVG 矢量图，可嵌入论文、报告、PPT。

**无需安装任何软件**，Safari / Chrome / Edge 均可。

---

## 快速上手

1. 双击 `index.html` 用浏览器打开
2. 点击顶栏预设按钮（如 **FBAR**、**InP EML**、**VOx IR** 等）加载示例
3. 点击 **⚡ 生成图表** 查看预览
4. 点击 **⬇ 导出 SVG** 保存

## 列说明

| 列名 | 说明 | 示例 |
|------|------|------|
| **#** | 序号（拖拽可排序） | 1, 2, 3... |
| **材料** | 材料名称，自动匹配色库 | `SiO2`, `VOx`, `InGaAsP` |
| **颜色** | 自定义颜色（点击色块选取） | 自动匹配，可手动改 |
| **y_bot** | 层底部坐标 (nm) | 0 |
| **y_top** | 层顶部坐标 (nm) | 500 |
| **Mask** | 掩模图案（定义该层覆盖区域） | `none`, `half`, `center`, `grating`... |
| **缩放** | 图像厚度缩放百分比（数据不变） | `100`=原样, `50`=视觉减半 |
| **字号** | 该层标签字号 (px) | `14` |
| **类型** | 沉积模式切换 | `Planar`（平坦）/ `Conformal`（共形） |
| **👁** | 隐藏/显示该层 | |
| **✕** | 删除该层 | |

## 掩模图案 (Mask)

| 图案 | 名称 | 效果 |
|------|------|------|
| `none` | 无 | 全覆盖 |
| `half` | 半区 | 左半 0-50% |
| `center` | 中心 | 35%-65% |
| `gate` | 栅极 | 28%-52% |
| `sti` | STI | 0-25% + 55%-100% |
| `mesa` | 台面 | 20%-60% |
| `fin` | Fin | 32%-38% + 48%-54% |
| `contact` | 接触孔 | 44%-48% |
| `via` | 通孔 | 45%-47% |
| `trench` | 沟槽 | 0-35% + 55%-100% |
| `ridge` | 脊波导 | 47%-53% |
| `gsg` | GSG 电极 | GND-Signal-GND 三指 |
| `via2` | 双通孔 | 30%-34% + 66%-70% |
| `grating` | 光栅 | 10 齿，50% 占空比 |

## 沉积类型

| 类型 | 行为 | 适用场景 |
|------|------|----------|
| **Planar** (平坦) | 顶面为水平面，填充台阶凹处 | PECVD SiO₂, SOG, CMP 后 |
| **Conformal** (共形) | 厚度均匀跟随下层表面轮廓 | ALD, LPCVD, 热氧化 |

---

## 缩放功能

「缩放」列控制该层在图像中的视觉厚度（百分比），**不改变实际数据**：

- `100` = 原始厚度（默认）
- `50` = 视觉上减半
- `20` = 压到 1/5

> 💡 用途：把很厚的牺牲层/衬底压扁，让薄的功能层更突出，同时标签仍显示真实厚度。

## 预设结构（11 种）

| 按钮 | 器件 | 关键特征 |
|------|------|----------|
| **FBAR** | 体声波谐振器 | Mo/AlN/Mo 压电堆栈 |
| **Accel** | MEMS 加速度计 | 牺牲层 + Poly-Si 质量块 |
| **Gyro** | MEMS 陀螺仪 | 锚点 + 驱动/检测质量 |
| **LNOI** | 铌酸锂光子 | LiNbO₃ 波导 + 电极 |
| **SiPH** | 硅光子 | SOI 波导 + Ge PD |
| **GaAs HEMT** | GaAs 高电子迁移率晶体管 | 异质结 + 肖特基栅 |
| **LNOI Mod** | LNOI 电光调制器 | 13层完整调制器结构 |
| **VOx IR** | 非制冷红外探测器 | 牺牲层 + VOx 热敏层 |
| **SAW** | 声表面波滤波器 | LT 衬底 + IDT 叉指 |
| **TC-SAW** | 温补 SAW | LT + IDT + SiO₂ 温补层 |
| **TF-SAW** | 薄膜 SAW | Si + 键合 SiO₂ + LT 薄膜 |
| **InP EML** | InP 电吸收调制激光器 | InGaAsP MQW + DFB 光栅 |

## 材料色库

输入材料名称时自动匹配颜色，支持中文/英文/化学式。已内置 45+ 种半导体材料：

`Si, SiO₂, SiN, Poly-Si, a-Si, Al, Cu, W, Ti, TiN, NiCr, Mo, TaN, Al₂O₃, HfO₂, VOx, TEOS, BPSG, PSG, SOG, PR, GaN, SiC, AlN, SiGe, Ge, CoSi₂, NiSi, ITO, ZrO₂, Ru, LiNbO₃, LiTaO₃, AlGaAs, GaAs, InP, InGaAs, InGaAsP, Au, Pt, Cr, Ni`

未匹配材料自动生成唯一颜色（基于名称哈希），也可手动用色块选取。

---

## SVG 导出

- Chrome/Edge：弹出系统原生保存对话框
- Safari：弹出命名窗口，输入文件名后下载
- 默认文件名：`stack_diagram.svg`
- SVG 背景透明，可直接嵌入文档

---

## 提示

- 拖拽行可调整膜层顺序
- 同名材料自动匹配相同颜色
- 点击 👁 可临时隐藏某层（不参与渲染）
- 缩放设为 1 以下可压扁厚层，突出薄层细节
